# Variables Aleatorias Discretas y Continuas

Este repositorio contiene ejemplos y aplicaciones prácticas de variables aleatorias en el ámbito empresarial.

## Estructura del Proyecto

- `1_variables_discretas.py` - Ejemplos con distribuciones Poisson y Binomial
- `2_variables_continuas.py` - Ejemplos con distribuciones Normal y Exponencial  
- `3_aplicacion_empresarial.py` - Aplicación práctica en optimización de inventarios
